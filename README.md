# GMS - Grading Management System

GMS is a microservices-based grading management platform with a web frontend, API gateway, and domain services for authentication, courses, marks, analytics, and policy management.

## What You Can Find Here
- Service-oriented backend organized by domain.
- Frontend app built in Next.js.
- Kubernetes manifests for deployment.
- Documentation for architecture, setup, development, and contribution.

## Quick Links
- Documentation Hub: [docs/README.md](docs/README.md)
- Blog Series: [Blog Series](https://yuvraj-rathod-1202.github.io/yuvraj.dev/blogs.html)

## Main Services
- `gateway`: API entry point and request routing
- `auth`: authentication and user management
- `courses`: course and role management
- `marks`: marks and grading data
- `analytics`: assessment analytics and background workers
- `policy`: policy and rule management
- `frontend`: user-facing web application
